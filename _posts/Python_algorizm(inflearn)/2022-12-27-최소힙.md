# 문제

![최소힙](../../images/2022-12-27-최소힙/최소힙.png)

# 풀이

```python
import sys
import heapq as hq
sys.stdin=open("input.txt","rt")
a=[]
while True:
    n=int(input())
    if n==-1:
        break
    if n==0:
        if len(a)==0:
            print(-1)
        else:
            print(hq.heappop(a))
    else:
        hq.heappush(a, n)
        


```

```
3
5
2
```
