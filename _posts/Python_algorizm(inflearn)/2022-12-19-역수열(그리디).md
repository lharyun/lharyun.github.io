# 풀이

```py
import sys
sys.stdin=open("input.txt","rt")

n=int(input())
a=list(map(int,input().split()))

seq=[0]*n

for i in range(n):
    for j in range(n):
        if a[i]==0 and seq[j]==0:
            seq[j]=i+1
            break
        elif seq[j] ==0:
            a[i]-=1
for x in seq:
    print(x ,end=' ')


cnt=0

for c in range(n):
    for d in range(n):
        if seq[d] == 0:
            cnt+=1
            

```

# 결과

4 8 6 2 5 1 3 7로 정상 출력됩니다.
