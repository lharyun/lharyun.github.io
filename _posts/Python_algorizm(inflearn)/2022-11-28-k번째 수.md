# 풀이

```
import sys
sys.stdin=open("input2.txt","rt")
T=int(input())
for t in range(T):
    n, s, e, k =map(int, input().split())
    a=list(map(int, input().split()))
    a=a[s-1:e]
    a.sort()
    print("#%d %d" %(t+1, a[k-1]))
```

# 결과
#1 7
#2 6
 으로 정상적으로 출력된다.
