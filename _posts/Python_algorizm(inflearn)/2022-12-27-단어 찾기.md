# 문제

![단어 찾기](../../images/2022-12-27-단어 찾기/단어 찾기.png)

# 풀이

```python
import sys
sys.stdin=open("input.txt","rt")

n=int(input())
p=dict()
for i in range(n):
    word=input()
    p[word]=1

for i in range(n-1):
    word=input()
    p[word]=0


for key, val in p.items():
    if val==1:
        print(key)
        break
    

```

blue 으로 정상 출력됩니다.
