#풀이 

```python
import sys
#sys.stdin=open('input.txt','rt')


n=int(input())

a=list(map(int,input().split()))

b=[0]*n
b[0]=a[0]
for i in range(1,n):
    if a[i]==1:
        if a[i-1]==1:
            b[i]=b[i-1]+1
        else:
            b[i]=1
    else:
        b[i]=0

print(sum(b))

```
# 결과
  10으로 정상적으로 출력된다.
