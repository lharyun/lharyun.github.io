# 풀이 




```python
import sys
sys.stdin=open("input.txt", "rt")
n=int(input())
mValue=0
for i in range(n):
    res=0
    a,b,c=map(int,input().split())
    if a==b and b==c:
        res=10000+(a*1000)
    elif (a==b and a!=c):
        res=1000+(a*100)
    elif (a==c and a!=b):
        res=1000+(a*100)
    elif (b==c and a!=b):
        res=1000+(b*100)
    else:
        res=100*max(a,b,c)
    if mValue<res:
        mValue=res

print(mValue)

```
# 결과
  12000으로 정상적으로 출력된다.
