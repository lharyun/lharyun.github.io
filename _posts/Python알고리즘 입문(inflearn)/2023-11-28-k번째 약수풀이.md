# k번째 약수 구하기

#풀이 

```python
n, k =map(int, input().split())<br>
cnt =0<br>
for i in range(1, n+1):
    if n%i==0:
        cnt+=1
    if cnt==k:
        print(i)
        break
else:
    print(-1)
```
