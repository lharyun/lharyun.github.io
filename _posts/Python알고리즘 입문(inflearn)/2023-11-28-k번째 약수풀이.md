# k번째 약수 구하기

#풀이 

```python
n, k =map(int, input().split())
cnt =0
for i in range(1, n+1):
    if n%i==0:
        cnt+=1
    if cnt==k:
        print(i)
        break
else:
    print(-1)
```
# 결과
  3으로 정상적으로 출력된다.
