#풀이 

```python
import sys
sys.stdin=open("input.txt","r")

n=int(input())
ch=[0]*(n+1)
cnt=0

for i in range(2, n+1):
    if ch[i]==0:
        cnt+=1
        for j in range(i,n+1,i):
            ch[j]=1
print(cnt)
              

```
# 결과
  23 73으로 정상적으로 출력됩니다.
