# 풀이

import sys
sys.stdin=open('input.txt','rt')


n=int(input())

for i in range(1,n+1):
    a=input()
    a=a.upper()
    if a==a[::-1]:
        print("#%d YES" %(i))
    else:
        print("#%d NO" %(i))

# 출력

```
#1 YES
#2 NO
#3 YES
#4 NO
#5 YES

```

으로 정상 출력된다.
